<p align='center'>Info-system is a library that provides you see basic information about your Operational System in a beautiful way!</p>

# Install

```node
$ npm i info-system
```

# Usage

<p>Currently have just a simple command line:</p>

```
$ info-system show
```

or, write with a shortcut

```
$ info-system s
```
